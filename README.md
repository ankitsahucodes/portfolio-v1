# Personal Portfolio Website

A modern, responsive personal portfolio website built using HTML5 and CSS3 to showcase my profile, skills, education, and projects in a clean dark-themed UI.

---
## Demo Link
[Live Demo](https://ankitsahuportfolio.vercel.app)


## Quick Start

```
git clone https://github.com/ankitsahucodes/portfolio-v1.git
cd portfolio-v1
npm install
npm run dev
```

## Technologies
- HTML5
- CSS3

## Features
- Responsive Design: Fully responsive layout that adapts seamlessly to desktop, tablet, and mobile devices
- Modern UI: Clean, dark-themed interface with smooth animations and hover effects
- Sticky Sidebar: Navigation sidebar that stays visible while scrolling (on desktop)
- Project Showcase: Grid layout displaying projects with images and tech stack tags
- Social Links: Quick access to GitHub, Twitter/X, and LinkedIn profiles
- Skills Display: Visual representation of technical skills with animated hover effects
